Dataset Information

Dataset: World Bank Population, total

Indicator: SP.POP.TOTL

Data units: Countries and economies with valid World Bank region metadata

Visualization: JIT Squarified TreeMap

Area encoding: Rectangle area represents population size.

Color encoding: Color represents the World Bank region using the $color field in the JIT JSON data.

Interaction Guide

Click a region or country/economy to zoom in and view details.

Right-click or use the Back button to zoom out.
